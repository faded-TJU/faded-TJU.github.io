Open Fault Data Benchmark

开放的故障数据集(OFDB)是现实的、大规模的和多样化的基准数据集，可用于故障诊断和故障预测等任务。OFDB数据集是使用OFDB Data Loader（后续封装）自动下载，处理和拆分的。可以使用OFDB评估器（后续封装）以统一的方式来评估模型性能。我们提供基于OFDB数据集的各类预训练模型用于故障诊断和预测应用。

The Open Fault Data Benchmark (OFDB) is a collection of realistic, large-scale, and diverse benchmark datasets for fault diagnosis or prediction. OFDB datasets are automatically downloaded, processed, and split using the OFDB Data Loader. The model performance can be evaluated using the OFDB Evaluator in a unified manner. We provide a variety of pre-training models based on OFDB datasets for fault diagnosis and prediction applications.
